\# 简易记账本 (Simple Accounting)



一个用 Python 写的命令行记账工具，支持分类统计和数据持久化。



\## 功能

\- 记录餐饮、交通、购物三类消费

\- 自动按日期保存为 CSV 文件

\- 支持异常输入处理（输错数字不会崩溃）

\- 支持历史记录加载



\## 技术栈

\- Python 3.x

\- 文件读写与异常处理

\- datetime 模块实现日期命名



\## 使用方法

1\. 运行 `main.py`

2\. 按提示输入金额和类别

3\. 输入 0 退出并自动保存
